High-level project understanding
- Editron is a browser-based code editor / cloud IDE built with Next.js App Router.
- It uses:
  - Next.js 15
  - React 19
  - TypeScript
  - Prisma with MongoDB
  - NextAuth
  - Monaco Editor
  - WebContainer API
  - xterm.js
  - Yjs and y-websocket for collaboration
  - AI SDK integrations for Gemini, Groq, and Mistral
- There is a main web app plus a standalone collaboration server in:
  - server/collab.ts
- The repo also includes a large starter-template catalog under:
  - editron-starters/
